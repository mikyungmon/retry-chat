### 채팅방

- 입력창 줄바꿈 입력 가능
- 메세지 리스트 출력 (줄바꿈 적용 필요)
  - 본인 오른쪽 (이름/아바타 미노출, 시간 분단위 노출)
  - 상대방 왼쪽 (이름/아바타/시간 분단위 노출)
  - 시스템 타입 왼쪽 (말풍선/이름/아바타 미노출)
    - 방을 생성하면, "000님이 방을 생성"
    - 방에 입장하면, "000님 입장"
    - 방에서 나가면, "000님 퇴장"
- 뒤로가기/퇴장하기 기능
